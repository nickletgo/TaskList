# TaskList
A replenisher task list

# Functional Requirements:
- [] As a user, I can create a task with following fields:
	* Task ID
	* Task Name
	* Description
	* CreatedBy 
	* AssignTo
	* CreatedOn
	* LastModifiedOn
	* DueOn
	* Priority
	* Effort
- [] As a user, I can view a list a task ordered by Priority and DueOn
